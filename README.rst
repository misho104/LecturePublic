Lecture Public
==============

Public materials for Sho's lectures.


For Students and Learners
-------------------------

Visit `the GitHub Pages <https://misho104.github.io/LecturePublic/>`_ for a more user-friendly interface to browse and download the materials.


Description
-----------

This repository contains documents and materials used in Sho's lectures, which hopefully are useful not only for the students in the courses but for students and educators from other places.

Feel free to use the materials under the CC BY-NC license! You can share and adapt them as long as you give appropriate credit, indicate if any changes were made, and don't use them for commercial purposes.

If you find any errors or have suggestions for improvements, please feel free to open an issue or submit a pull request.



Contents
--------

For general learners in Engineering and Science,

- |GeneralPhysics|_: `General Physics <https://www2.nsysu.edu.tw/iwamoto/gp1.html>`_ lectures for university first-year students in engineering and science.

For students enrolled in Sho's courses,

- |Policies|_: Guidelines regarding Sho's lectures.
- |PastLectureAdmin|_: Administrative documents for past lectures.
